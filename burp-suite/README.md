# Burp Suite Advanced

## Objective
Analyze and manipulate web traffic.

## Tasks Performed
- Intercepted login requests  
- Modified parameters  
- Used Intruder for fuzzing  

## Steps
1. Enabled proxy in browser  
2. Captured request in Burp  
3. Sent to Intruder  
4. Performed payload-based attack  

## Impact
- Identify vulnerabilities  
- Test authentication flaws  

## Mitigation
- Strong input validation  
- Rate limiting  
- Secure authentication mechanisms
