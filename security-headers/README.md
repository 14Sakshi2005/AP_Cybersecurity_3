# Web Security Headers

## Objective
Improve application security using HTTP headers.

## Analysis
- Used securityheaders.com  
- Identified missing headers  

## Implemented Headers
- Content-Security-Policy  
- X-Frame-Options  
- X-Content-Type-Options  
- Strict-Transport-Security  

## Impact
- Prevent XSS & clickjacking  
- Enforce secure communication  

## Mitigation
- Configure headers in Apache server  
- Regular security audits
